## Custom Badges & Title

```tsx showLineNumbers
<RestApi
  circleText="API"
  title="Custom REST Flow"
  badgeTexts={{
    first: "GET",
    second: "POST",
    third: "PATCH",
    fourth: "REMOVE",
  }}
  buttonTexts={{
    first: "MyTeam",
    second: "api_updates",
  }}
/>
```

---

## Props

| Prop          | Type                                                               | Default                                       | Description                                      |
| ------------- | ------------------------------------------------------------------ | --------------------------------------------- | ------------------------------------------------ |
| `className`   | `string`                                                           | `-`                                           | Extra CSS classes                                |
| `circleText`  | `string`                                                           | `"SVG"`                                       | Text inside the bottom circle                    |
| `badgeTexts`  | `{ first: string; second: string; third: string; fourth: string }` | `{ GET, POST, PUT, DELETE }`                  | Labels for the top API method badges             |
| `buttonTexts` | `{ first: string; second: string }`                                | `{ LegionDev, v2_updates }`                   | Labels inside the animated bottom badges         |
| `title`       | `string`                                                           | `"Data exchange using a customized REST API"` | Box header text                                  |
| `lightColor`  | `string`                                                           | `"#00A6F5"`                                   | The highlight glow color used in animated lights |
