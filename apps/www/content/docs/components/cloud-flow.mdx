## Custom Badges & Title

```tsx showLineNumbers
<CloudFlow
  centerText="CORE"
  title="Distributed Cloud Architecture"
  nodeLabels={{
    topLeft: "API",
    topRight: "CDN",
    bottomLeft: "Database",
    bottomRight: "Worker",
  }}
  badges={{
    left: "Live Traffic",
    right: "Sync Active",
  }}
  accentColor="#22C55E"
/>
```

---

## Props

| Prop          | Type                                                                             | Default                            | Description                                 |
| ------------- | -------------------------------------------------------------------------------- | ---------------------------------- | ------------------------------------------- |
| `className`   | `string`                                                                         | `-`                                | Extra CSS classes                           |
| `centerText`  | `string`                                                                         | `"HUB"`                            | Text inside the center hub circle           |
| `title`       | `string`                                                                         | `"Distributed Cloud Architecture"` | Header badge title                          |
| `accentColor` | `string`                                                                         | `"#10B981"`                        | Accent color for glow, icons, and gradients |
| `nodeLabels`  | `{ topLeft: string; topRight: string; bottomLeft: string; bottomRight: string }` | `{ API, CDN, DB, Worker }`         | Labels for the cloud nodes                  |
| `badges`      | `{ left: string; right: string }`                                                | `{ Live Traffic, Sync Active }`    | Floating info badges                        |

- ✔ **Animated SVG cloud flow paths**
